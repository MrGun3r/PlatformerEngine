# Compilation
To compile this run "make" in your terminal / command prompt or run :<br />
          for windows:<br />
                    ```gcc main.c -lmingw32 -lSDL2main -lSDL2 -lSDL2_image -lSDL2_mixer -o Game```<br />
          for linux:<br />
                    ```gcc main.c -lSDL2main -lSDL2 -lSDL2_image -lSDL2_mixer -o Game```
